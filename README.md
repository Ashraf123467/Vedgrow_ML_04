# 🧠 CNN Image Classification App

A Deep Learning based Handwritten Digit Recognition System developed using Convolutional Neural Networks (CNN) and the MNIST dataset.  
This project classifies handwritten digits (0–9) with high accuracy 

---

# 🚀 Project Overview

This project demonstrates the implementation of a CNN model for image classification using the MNIST handwritten digit dataset.  
The model is trained to recognize handwritten digits and predict the correct digit from uploaded custom images.

The project also includes:
- Data preprocessing
- Exploratory Data Analysis (EDA)
- CNN model training
- Model evaluation
- Visualization of results

---

# 🎯 Objectives

- Build a CNN model for handwritten digit classification
- Understand image preprocessing techniques
- Analyze the MNIST dataset using EDA
- Visualize model accuracy and loss
- Evaluate model performance using confusion matrix and classification report

---

# 🧾 Dataset Used

## MNIST Dataset

The MNIST dataset contains:
- 70,000 grayscale handwritten digit images
- Image size: 28 × 28 pixels
- 10 output classes (Digits 0–9)

Dataset Split:
- Training Images: 60,000
- Testing Images: 10,000

# 🧠 CNN Architecture

The Convolutional Neural Network consists of:

- Convolutional Layers
- MaxPooling Layers
- Batch Normalization
- Dropout Layers
- Flatten Layer
- Dense Layers
- Softmax Output Layer

The architecture helps the model learn image patterns and classify digits efficiently.

- The CNN model was tested on custom handwritten digit images using Google Colab.
